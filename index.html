RecGrab Privacy Policy

Last updated: June 12, 2026

RecGrab does not collect, transmit, sell, or share personal data.

Data Stored by the Extension

RecGrab stores user preferences using Chrome extension storage so the extension can work consistently when a user returns to Recreation.gov. These preferences may include:

  • Selected Recreation.gov entry points
  • Configured group size
  • Target date
  • Filtering preferences
  • Global on and armed settings

These settings are used only to operate the extension and are not sent to an external server by RecGrab.

Recreation.gov Page Access

RecGrab reads Recreation.gov page content in the user's browser only to filter availability, highlight matching entries, set group size, center the configured target date, and assist with selecting matching availability.

This page content is processed locally in the browser. RecGrab does not send Recreation.gov page data to any external service.

No Tracking or Advertising

RecGrab does not use analytics, advertising, tracking pixels, behavioral profiling, or third-party data sharing.

Contact

For privacy questions, contact the developer at kevinjin420+recgrabcrx@example.com.

This policy applies to the RecGrab Chrome extension.
